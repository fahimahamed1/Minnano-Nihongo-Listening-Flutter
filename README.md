## ğŸš€ Getting Started

### Prerequisites
- Flutter SDK 3.0.0+
- Dart SDK 3.0.0+

### Installation

```bash
# Clone the repository
git clone https://github.com/fahimahamed1/Minnano-Nihongo-Flutter.git

# Navigate to project
cd Minnano-Nihongo-Flutter

# Install dependencies
flutter pub get

# Run the app
flutter run
```

### Build Release

```bash
# Android APK
flutter build apk --release

# Android App Bundle
flutter build appbundle --release

# iOS (macOS only)
flutter build ios --release
```
